# 招聘数据漏斗规则

> Phase 4 记录，Phase 5-11 逐步实现。

## 漏斗体系

系统覆盖 8 大漏斗维度，不限于数量统计：

1. 岗位漏斗 — 岗位→候选人→投递→面试→Offer→入职
2. 候选人漏斗 — 来源渠道→阶段分布
3. 投递阶段漏斗 — 各阶段停留时长+卡点
4. 面试漏斗 — 通过率+反馈质量
5. Offer漏斗 — 风险+转化
6. 渠道漏斗 — ROI分析
7. 面试官反馈漏斗 — 质量评估
8. 招聘周期漏斗 — 端到端时效

## 当前 Phase 4 能力

- Candidate/Application 数据链路已建立
- 投递阶段（12阶段）已可追踪
- 后续 Phase 在此基础上扩展聚合分析
