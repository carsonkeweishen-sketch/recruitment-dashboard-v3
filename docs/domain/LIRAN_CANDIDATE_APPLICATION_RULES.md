# 候选人/投递规则

> Phase 4 使用。后续 Phase 5/6/9/10/11 扩展。

## 招聘漏斗体系（后续 Phase）

系统支持详细多维招聘漏斗，核心指标：

| 漏斗类型 | 核心指标 |
|----------|----------|
| 岗位漏斗 | 入库数→HR通过率→业务通过率→面试通过率→Offer转化率→入职转化率 |
| 候选人漏斗 | 来源渠道分布→阶段分布→转推率 |
| 投递阶段漏斗 | 各阶段停留时长→卡点数量 |
| 面试漏斗 | 初试/复试/终面通过率→反馈完成率→证据充分率 |
| Offer漏斗 | Offer风险数→竞品Offer→薪资差距→入职前失联 |
| 渠道漏斗 | 各渠道ROI→各渠道转化率 |
| 面试官反馈漏斗 | 反馈及时性→评分一致性→证据质量 |
| 招聘周期漏斗 | 岗位平均招聘周期→各阶段平均时长 |

当前 Phase 4 仅实现 Candidate/Application 基础链路，为后续漏斗提供数据基础。

## 联系方式权限控制

- admin/leader/hrbp: 可查看手机/邮箱
- recruiter: 可查看自己负责候选人的联系方式
- business_owner/interviewer: 不可查看联系方式
- 列表页默认不展示联系方式，仅在 Drawer 详情中按权限展示
