# 面试表现偏差风险规则

> Phase 4 记录，Phase 6/10 实现。

## 风险识别信号

系统识别但不自动决策以下信号：

- 评分高但证据文本很空
- 沟通表达分高但岗位胜任力证据弱
- 反馈只写主观感受，缺少事实行为
- 多轮面试评分波动异常
- 面试官结论强烈但证据不足
- 候选人经历与岗位关键场景不匹配

## 产品表达

- 不使用"面霸"等非专业用语
- 使用"面试表现偏差风险""证据不足风险"等专业表达
- 系统仅作风险提醒+追问建议，不自动淘汰/推进

## 落地计划

- Phase 6: 结构化面试反馈+证据记录
- Phase 10: AI面试表现分析
- Phase 11: 复盘报告中的风险摘要
