# Phase 8.14 CEO Demo Script

## 开场 30 秒
"这个工具解决的是招聘中最痛的判断问题：岗位到底要什么人、面试有没有问到关键证据、候选人是否真的匹配、流程卡在哪里。现在我们把理然真实JD、岗位SOP、AI助手和行动中心串起来了。"

## Dashboard
"首页展示100个真实岗位。候选人为0是因为还没接入真实招聘数据——系统不造假，数据不足就诚实显示。右上角是AI助手入口。"

## AI Copilot
"AI是全局能力，根据当前页面上下文回答。问它'今天有哪些风险'，如果证据充分会给出带引用的建议。注意：provider、model、引用来源全可见。Human Review三态——接受、编辑后接受、忽略。Draft Action只是草稿，必须人工确认。"

## Funnel
"漏斗展示真实数据。数据不足时明确标注partial，不把null当0。瓶颈阶段高亮，点击可进入Action。"

## Action Center
"卡点不是只展示，而是转成行动闭环。AI生成的Action草稿必须人工确认后才创建。"

## Job Detail
"打开真实岗位详情。JD原文、职责、source_file/source_sheet/source_row全可追溯。证明不是假岗位。"

## Knowledge
"搜索岗位关键词，命中真实JD和SOP chunk。AI的所有建议都基于这些资料，引用来源可追溯。"

## 结尾
"当前状态：真实数据已接入、AI可信可追溯、招聘判断闭环已形成。后续需要真实候选人数据、Moka/飞书集成。"
