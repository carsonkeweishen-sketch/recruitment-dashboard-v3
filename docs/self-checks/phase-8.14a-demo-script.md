# Phase 8.14A CEO Demo Script (5分钟)

## 开场
"这个工具不是替代Moka，而是招聘判断层+AI分析层+风险闭环层。我们把理然真实JD、岗位SOP、AI助手和行动中心串起来了。"

## Dashboard
"首页100个真实岗位。候选人为0是还没接入真实数据——系统不造假。"

## AI Copilot
"AI根据当前页面上下文回答。provider、model、引用来源全可见。Human Review三态。Draft Action只是草稿。证据不足时不瞎答。"

## Funnel
"漏斗展示真实数据。不足时标注partial。瓶颈阶段高亮。"

## Action Center
"卡点转成行动闭环。AI生成的Action草稿必须人工确认。"

## Job Detail
"真实岗位详情。JD原文、职责、source追溯全可读。"

## Knowledge
"搜索命中真实JD和SOP chunk。AI的建议都基于这些资料。"

## 结尾
"真实数据已接入、AI可信、判断闭环已形成。后续需要真实候选人数据。"
