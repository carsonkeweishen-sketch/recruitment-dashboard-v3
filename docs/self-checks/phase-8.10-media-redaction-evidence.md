## 脱敏时机

| 时机 | 是否执行脱敏 | 说明 |
|------|-------------|------|
| Transcript 存储前 | ✅ 是 | RedactionService 在写入 DB 前执行 |
| Transcript Segment 存储前 | ✅ 是 | 每个 Segment 文本单独脱敏 |
| Transcript API 响应 | ✅ 是 | 再次校验，双重保障 |
| AI 分析输入 | ✅ 是 | 分析前脱敏，防止 LLM 接触 PII |
| Activity Log 写入 | ✅ 是 | 日志中的文本片段也脱敏 |
| 前端渲染 | ✅ 是 | 前端展示前再次校验 |
