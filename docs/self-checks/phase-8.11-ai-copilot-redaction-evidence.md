## 脱敏规则

| 类别 | 正则/规则 | 替换为 | 示例 |
|---|---|---|---|
| 手机号 | `1[3-9]\d{9}` | `[手机号已脱敏]` | `13800138000` → `[手机号已脱敏]` |
| 邮箱 | `[\w.-]+@[\w.-]+\.\w+` | `[邮箱已脱敏]` | `zhangsan@company.com` → `[邮箱已脱敏]` |
| 身份证号 | `\d{17}[\dXx]` | `[身份证号已脱敏]` | `110101199001011234` → `[身份证号已脱敏]` |
| 薪资 | `(月薪\|年薪\|薪资\|工资)\s*[:：]?\s*\d+[kKwW万]?` | `[薪资信息已脱敏]` | `月薪: 25K` → `[薪资信息已脱敏]` |
| API Key | `sk-[a-zA-Z0-9]{20,}` | `[API_KEY]` | `sk-abc123...` → `[API_KEY]` |

## 边界场景验证

| 场景 | 输入 | 预期 | 实际 | Verdict |
|---|---|---|---|---|
| 已脱敏字符串再次脱敏（幂等） | `[手机号已脱敏]` | 不重复脱敏 | 输出不变 | PASS |
| 混合字段 | "姓名张三 手机13912345678 邮箱a@b.com" | 姓名保留，手机和邮箱脱敏 | 姓名保留，PII 脱敏 | PASS |
| 无 PII 数据 | "今天天气很好" | 原样返回 | 原样返回 | PASS |
| 空字符串 | "" | "" | "" | PASS |
| 部分匹配（非手机号 12 位数字） | "订单号 123456789012" | 不触发脱敏 | 原样保留 | PASS |
